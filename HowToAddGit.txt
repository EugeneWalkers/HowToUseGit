creating repository in GitHub

go to directory
0.1 git init (creating .git)

0.2 git remote add <name> https://gihub.com/.../directory.git  (to add remote on github)
    example:
    git remote add origin https://github.com/EugeneWalkers/HowToUseGit.git
    
git remote -v (for looking)

1. git add . (adding all files from local diretory)

2. git commit -m "Some changes"

3. git push origin master (push = push, origin = name remote's, master = branch)

git status (changes from the last commit)

git rm "file" (for deleting file)
